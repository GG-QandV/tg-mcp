# Telegram MCP — Полный список инструментов

## ЧАТЫ (Chats)

| Инструмент | Описание |
|---|---|
| **GetAllChats** | Получить все чаты (группы и каналы), в которых состоит пользователь |
| **GetChats** | Получить список всех диалогов (с лимитом, архивированные, по папкам) |
| **GetChatInfo** | Получить детальную информацию о чате (ID, тип, название, участники, описание) |
| **GetChatMembers** | Получить участников чата или канала (с фильтром: recent/admins/kicked/bots) |
| **GetChatAdmins** | Получить администраторов чата или канала |
| **GetChatOnlineCount** | Получить количество онлайн-участников в чате |
| **SearchChats** | Поиск чатов по названию или username |
| **GetChatInviteLink** | Получить пригласительную ссылку для чата |
| **CheckChatInvite** | Проверить информацию о пригласительной ссылке |
| **JoinChatByInvite** | Присоединиться к чату по пригласительной ссылке |
| **LeaveChat** | Покинуть чат или канал |
| **GetFolders** | Получить папки чатов (диалоговые фильтры) |
| **GetChatsFromFolder** | Получить чаты из указанной папки |
| **GetForumTopics** | Получить список тем форума в чате |
| **AddChatMember** | Добавить участника в чат |
| **BanChatMember** | Забанить участника в чате (с опцией удаления истории) |
| **UnbanChatMember** | Разбанить участника в чате |
| **KickChatMember** | Выгнать участника из чата |
| **PromoteToAdmin** | Назначить участника администратором (с настройкой прав) |

## СООБЩЕНИЯ (Messages)

| Инструмент | Описание |
|---|---|
| **SendMessage** | Отправить текстовое сообщение в чат (с поддержкой HTML/Markdown, тихий режим, отложенная отправка) |
| **ReplyToMessage** | Ответить на конкретное сообщение |
| **EditMessage** | Редактировать отправленное сообщение |
| **DeleteMessage** | Удалить одно или несколько сообщений (с опцией revoke) |
| **ForwardMessage** | Переслать сообщения из одного чата в другой (в т.ч. как альбом) |
| **GetChatHistory** | Получить историю сообщений чата (с пагинацией по offset_id/min_id/max_id) |
| **SearchMessages** | Поиск сообщений в чате или глобально (с фильтром: photo/video/document/url/voice/music/gif) |
| **MarkAsRead** | Отметить сообщения как прочитанные |
| **PinMessage** | Закрепить сообщение в чате |
| **UnpinMessage** | Открепить сообщение (или все сообщения) из чата |
| **SendFile** | Отправить файл в чат (с опциональной подписью и reply) |
| **DownloadMedia** | Скачать медиа-файл из сообщения на диск |

## КОНТАКТЫ (Contacts)

| Инструмент | Описание |
|---|---|
| **GetContacts** | Получить список контактов пользователя |
| **AddContact** | Добавить новый контакт (по номеру телефона и имени) |
| **DeleteContact** | Удалить контакт |
| **SearchContacts** | Поиск контактов по имени или username |
| **BlockUser** | Заблокировать пользователя |
| **UnblockUser** | Разблокировать пользователя |
| **GetBlockedUsers** | Получить список заблокированных пользователей |
| **SearchGlobal** | Глобальный поиск пользователей и чатов |

## ПОЛЬЗОВАТЕЛИ (Users)

| Инструмент | Описание |
|---|---|
| **GetMe** | Получить информацию о текущем пользователе (ID, имя, телефон, премиум, верификация) |
| **GetUserInfo** | Получить детальную информацию о пользователе (включая bio, общие чаты, статус блокировки) |
| **GetUserStatus** | Получить онлайн-статус пользователя (online/offline/was-recently) |
| **SearchUsers** | Поиск пользователей по имени |
| **ResolveUsername** | Найти пользователя или чат по username (@username) |
| **UpdateProfile** | Обновить профиль (имя, фамилию, описание "о себе") |

## ГРУППЫ / КАНАЛЫ (Groups & Channels)

| Инструмент | Описание |
|---|---|
| **CreateGroup** | Создать новую группу с указанными участниками (с опциональным описанием) |
| **CreateChannel** | Создать новый канал или супергруппу (с описанием) |
| **EditChatTitle** | Редактировать название чата/канала/группы |

## ВХОДЯЩИЕ (Inbox)

| Инструмент | Описание |
|---|---|
| **InboxPeek** | Проверить количество входящих сообщений без их чтения (неблокирующий) |
| **InboxRead** | Прочитать все накопленные входящие сообщения (буфер очищается после чтения) |

---

**Всего инструментов: 50**
**Протокол:** MCP (Model Context Protocol), transport — stdio
**Библиотека:** Telethon (MTProto, push-уведомления)
